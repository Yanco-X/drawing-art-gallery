# Personal Art Gallery

This is a personal project which serves as a personal art gallery to upload and display drawings. This project is done for learning purposes, and does not intend to generate any type of monetization, sales, or similar. This will be a project to practice, learn, and develop coding skills as well as expand knowledge.

## Tech Stack

**Backend:**
- Python
- Flask (REST API)
- PostgreSQL

**Frontend:**
- React
- TypeScript
- Tailwind CSS

## Roadmap

**Phase 1: Local Storage**
- Set up the basic foundation for the project.
- Implement the backend and frontend.
- Initially load and display pictures directly from the local disk.

**Phase 2: Cloud Storage**
- Plan and implement a strategy to store pictures online (e.g., Cloudinary, AWS S3, Google Cloud Storage, or others).
- Grab and serve images from the cloud storage instead of the local disk.
